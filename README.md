# 中醫嘉年華｜體質探索站 V8.1.1 正式版

## 上傳 GitHub Pages
請將以下內容全部上傳到同一個 repository 的根目錄：

- `index.html`
- `style.css`
- `script.js`
- `assets/` 整個資料夾

圖片路徑已設定為 `assets/檔名`，請不要把圖片放成 `assets/assets/檔名`。

## V8.1.1 更新內容

1. 第一階段改為：每種體質 2 題合計 ≥5 分才進入該體質第二階段。
2. 若八種偏性體質皆未達 5 分，直接進入平和體質問卷。
3. 平和體質維持 ≥60 分成立，並保留反向計分。
4. 結果頁新增「您的體質夥伴」。
5. 多體質結果每一種體質皆顯示完整說明與建議。
6. 加入低調 Footer 作者署名。
7. 圖片顯示尺寸與首頁版面微調。
